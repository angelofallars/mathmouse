# mathmouse
A Discord math quiz bot made using Python.

# Requiremenets
`python3`

# Installation

```bash
git clone https://github.com/angelofallars/mathmouse
cd mathmouse
pip install -r ./requirements.txt
```

To run:

```bash
python3 __main__.py
```

# Usage

```Available quizzes:
m.quiz addition
m.quiz subtraction
m.quiz multiplication
m.quiz division

Display your score
m.score

Display someone else's score
m.score <user>```
